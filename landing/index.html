Features

Everything from your meeting.
Nothing you didn't ask for.

Always up to date

Syncs with Granola every 30 seconds. Action items and follow-ups appear in the panel the moment your meeting ends.

Sorted by Claude

Claude reads your notes and separates tasks you need to complete from follow-ups you need to send — automatically.

Notes always there

Every item links to its source. Click the notes icon on any task to read the full meeting summary right alongside your list.

Emails in seconds

Draft follow-up emails in a click. Oats pre-fills the recipient, subject, and a personal greeting drawn from the meeting context.

Pure keyboard flow

Navigate with arrow keys, complete with ↵, open Granola with G. Run the whole workflow without touching your mouse.

Claude Code ready

Press C on any task and Oats passes it to Claude Code with the full meeting notes as context — ready to start working immediately.

Terminal

A full CLI for people who live in the terminal.

Run oats for an interactive TUI with arrow-key navigation, live sync, notes access, and Claude Code handoff — no mouse required.

Pipe-friendly oats list and oats sync are available for scripting.
